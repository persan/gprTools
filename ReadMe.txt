gpr_tools:
This project contains small utilities to make it simpler to work with
.gpr files and makefiles in concert-

 - Source_Dirs
 - Main
 - Object_Dir(s)
 - Exec_Dir(s)
 - Languages
 - Imported projects
 - Missing imports.
 from projectfiles on standard output.
